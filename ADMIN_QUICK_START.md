## ğŸ”§ Bulk Operations Guide

### Select Multiple Cars
- Click checkbox in table header = Select ALL
- Click individual checkboxes = Select specific cars
- Click **"Clear"** to deselect all

### Available Bulk Actions
1. **Set Start Date**: Apply same start time to all selected
2. **Set End Date**: Apply same end time to all selected
3. **Enable Bidding**: Make all selected visible to users
4. **Disable Bidding**: Hide all selected from users

### Best Practice
1. Select cars in same category/location
2. Set appropriate date ranges
3. Enable in batches
4. Monitor for first few bids
